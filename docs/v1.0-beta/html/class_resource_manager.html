Detailed Description

Play an audio file.

Parameters
audioTypeType of the audio file to play
filenameFilename of the audio to play
isLoopedTrue if audio should be looped, otherwise false
Exceptions
FileNotFoundif the audio file cannot be found on the disk

The audio file are played in a separate thread. This means that, the main thread is not blocked and other audio file may be played simultaneously.

Note
It's much faster to preload the audio files with the having the audio manager load each individual file from the disk before playing it
