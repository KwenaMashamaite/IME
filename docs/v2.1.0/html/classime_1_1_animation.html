Detailed Description

A SpriteSheet based animation.

An ime::Animation specifies how a ime::Sprite object is animated.

In order to play the animation it must be added to a ime::Animator

◆ setRepeatCount()

void ime::Animation::setRepeatCount ( int  count)

Set the repetition count of the animation.

Parameters
countThe number of times to repeat the animation

Set count to '-1' to repeat the animation forever or set it to some positive number x, where x is the number of times to repeat the animation before it completes or set it to '0' to disable repetition if the animation is currently repeating.

Note that if count is set to a negative value that is not '-1', then the repeat count will be set to '0', meaning that the animation will not repeat

By default, the animation does not repeat (repeat count is 0), it plays once then completes

◆ setTimescale()

void ime::Animation::setTimescale ( float  timescale)

Set the timescale factor of the animation.

Parameters
timescaleThe new timescale factor

The timescale factor controls the playback speed of the animation. For example, a timescale of 2.0f will make the animation play twice as fast, a timescale of 0.5f will make the animation play at half its normal playback speed and a timescale of 0.0f will freeze the animation. Note that if timescale is negative then the timescale will be set to 1.0f (real-time)

By default the timescale is 1.0f (real-time)
