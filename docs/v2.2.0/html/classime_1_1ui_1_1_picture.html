Create a picture from a certain parts of an image.

Parameters
filenameFilename of the image to construct the picture from
frameFrame to load from source image as picture
transparentTextureTrue to pass mouse events to the widget behind the picture on transparent parts of the picture, otherwise false
Exceptions
FileNotFoundIf the image cannot be found on the disk
Returns
The new picture widget

The frame is in the form {top, left, width, height}

By default, the picture is the same size as the last arguments of the frame

Picture("image.png", {20, 15, 60, 40}}); // Load part of the image from (20,15) to (80,55)
