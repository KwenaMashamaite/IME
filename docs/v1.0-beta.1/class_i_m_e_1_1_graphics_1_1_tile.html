◆ setSpritePosition()

void IME::Graphics::Tile::setSpritePosition ( float  x,
float  y 
)

Set the position of the sprite within the tile.

Parameters
xX coordinate of the sprite
yY coordinate of the sprite

The sprite will be moved to the new position immediately. Nothing will happen if the specified position lies outside the tile borders.

Note
This function will not check if the entire sprite lies in within the borders of the tile, it only checks if the sprites origin lies within the tile borders or not. The sprite has the same position as the tile by default
