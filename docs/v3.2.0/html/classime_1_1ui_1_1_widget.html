Detailed Description

An Abstract Base Class for Graphical User Interface (GUI) elements.

Events generated by ime::ui::Widget
Name Argument Type Description
focus void The widget is focused
unfocus void The widget lost focused
mouseEnter void The mouse cursor entered the widget
mouseLeave void The mouse cursor left the widget
animationFinish void The widgets animation finished playing
nameChange std::string The name of the widget changed. Parameter = The new name of the widget
sizeChange ime::Vector2f The size of the widget changed. Parameter = The new size of the widget
positionChange ime::Vector2f The position of the widget changed. Parameter = The new position of the widget

Usage Example:

widget.on("mouseEnter", ime::Callback<>([] {
std::cout << "Mouse entered widget" << "\n";
}));
widget.on("mouseEnter", ime::Callback<ime::ui::Widget*>([](ime::ui::Widget* widget) {
std::cout << widget->getName() << "\n";
}));
widget.on("positionChange", ime::Callback<ime::Vector2f>([](ime::Vector2f pos) {
std::cout << "Widget moved to {" << pos.x << ", " << pos.y << "}" << "\n";
})):

Set the local origin of the widget.

originNew origin

The origin of the widget defines the center point for all transformations (position, scale, rotation). The coordinates of this point must be relative to the top-left corner of the widget, and ignore all transformations (position, scale, rotation).
