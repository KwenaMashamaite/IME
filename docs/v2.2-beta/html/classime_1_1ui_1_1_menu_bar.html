◆ addMenuItem() [1/3]

void ime::ui::MenuBar::addMenuItem ( const std::initializer_list< std::string > &  hierarchy)

Add a new menu or a sub menu item.

Parameters
hierarchyHierarchy of the menu item, starting with the parent menu and ending with menu item to be added

Note that the hierarchy must contain at least two items. The first item is the name of the parent menu and the second item is the sub-menu of the parent menu. Items three and beyond are cascades of the second item. If the parent menu does not exist it will be created

menuBar->addMenuItem({"File", "Save"});
menuBar->addMenuItem({"File", "Open recent", "SaveGame01"});
menuBar->addMenuItem({"File", "Open recent", "SaveGame02"});
menuBar->addMenuItem({"Window", "Editor tabs", "Split", "Left"});
