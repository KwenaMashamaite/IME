Detailed Description

A widget that can be used to graphically visualize the progress of an operation.

Events generated by ime::ui::ProgressBar
Name Argument Type Description
valueChange unsigned int The value of the progress bar changed. Parameter = new value
full void The progress bar became full

Event usage example:

progressBar.on("full", ime::Callback<>([]() {
std::cout << "Loading complete" << "\n";
}));

◆ FillDirection

enum ime::ui::ProgressBar::FillDirection
strong

Determines in which direction the progress bar is filled when value goes from minimum to maximum.

Enumerator
LeftToRight 

Start filling at the left side and go to the right.

RightToLeft 

Start filling at the right side and go to the left.

TopToBottom 

Start filling at the top an go downward.

BottomToTop 

Start filling at the bottom and go upward.
