\doxysubsection{Detailed Description}
Intermediate Abstract Base Class (A\+BC) for game entities. 

This class implements the domain object interface (\mbox{\hyperlink{class_i_entity}{I\+Entity}}). All methods that are the same for all game entities (such as getters and setters) are implemented by this class. Those that are entity specific are passed down to derived classes to implement. As a result, the class itself is an A\+BC. All game entities inherit directly from this class instead of \mbox{\hyperlink{class_i_entity}{I\+Entity}} to avoid code duplication within derived classes (D\+RY).

\begin{DoxyNote}{Note}
All methods implemented by this class are not final. This means that they can be overridden by derived classes. 
\end{DoxyNote}
