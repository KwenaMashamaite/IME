Set the local origin of the slider.

Parameters
xX coordinate of the new origin
yY coordinate of the new origin

The origin of the slider defines the center point for all transformations (position, scale, rotation). The coordinates of this point must be relative to the top-left corner of the slider, and ignore all transformations (position, scale, rotation).

The default origin of the slider is (0, 0)
